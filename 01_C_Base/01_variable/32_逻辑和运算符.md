### 短路

![image-20260331225458016](https://raw.githubusercontent.com/czh010/images/main/hardwa/image-20260331225458016.png)

```c
#include <stdio.h>

int main(){
    int a = -1;
    if ( a>0 && a++>1){
        printf("OK\n");
    }
    printf("%d\n",a);
    return 0;
}
```
