# *数组的例子*

## 应用题

![image-20260402150504169](https://raw.githubusercontent.com/czh010/images/main/hardwa/image-20260402150504169.png)

```c
#include <stdio.h>

int main(void)
{
    const int number = 10;
    int x;
    int count[number];
    int i;

    for ( i=0; i<number; i++ ) {
        count[i] = 0;
    }
    scanf("%d", &x);
    while ( x!= -1 ) {
        if ( x>=0 && x<=9 ) {
            count[x] ++;
        }
        scanf("%d", &x);
    }
    for ( i=0; i<number; i++ ) {
        printf("%d:%d\n", i, count[i]);
    }
    return 0;
}
```
